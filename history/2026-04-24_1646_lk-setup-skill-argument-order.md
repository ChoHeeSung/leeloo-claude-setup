# lk-setup 스킬 Argument-Hint 순서 및 Model 미리보기 개선

## 작업 개요

`leeloo-kit/skills/lk-setup` 스킬의 SKILL.md 업데이트. 사용자가 스킬을 호출할 때 자동완성 UI에서 필요한 정보를 명확하게 제시하고, 모델 선택 옵션에 대한 미리보기 정보를 추가.

## 변경 내용

### 1. Argument-Hint 순서 변경
- **배경**: 모델 파라미터가 argument-hint의 마지막 위치에 있어서 자동완성 UI에서 잘려나갈 수 있음
- **개선**: `model` 파라미터를 세 번째 위치로 이동
- **효과**: 사용자가 자동완성 팝업에서 필수 파라미터를 모두 확인 가능

### 2. Model 선택 화면에 Preview 필드 추가
- **대상 모델**:
  - Opus 4.7
  - Sonnet 4.6
  - Sonnet 4.6 1M (1M context)
  - Haiku 4.5
  
- **각 모델별 표시 정보**:
  - 모델 ID (예: `claude-opus-4-1-20250805`)
  - 컨텍스트 윈도우 크기 (예: `200k tokens`)
  - 속도 및 성능 지표 (예: `Very Fast` / `Cost-Effective`)
  - 모델의 핵심 특성 (예: `일반용 추론`, `비용 최적화`, `장문맥 처리`)
  - 추천 사용 작업 (예: `코드 분석`, `복잡한 문제 해결`, `데이터 처리`)

### 3. 사용 흐름 개선
사용자가 `/lk-setup` 스킬을 실행할 때:
1. 환경 강화 도구의 주요 기능 목록 제시
2. 모델 선택 프롬프트에서 각 모델의 미리보기(preview) 정보 시각화
3. 모델 선택 후 설치/업데이트 진행
