# lk-setup-argument-hint-sync

## 지시 요약

`leeloo-kit`의 `lk-setup` 명령어에 새로 추가된 기능(커밋 `eaee13c`)을 문서에 반영하여 최신 상태로 동기화.

## 작업 내용

### 변경 파일
- `leeloo-kit/commands/lk-setup.md`

### 수정 사항

1. **description 업데이트**
   - 기존: `환경 강화 도구(설치·statusline·플러그인 토글)`
   - 변경: `환경 강화 도구(설치·statusline·플러그인 토글·기본 모델 설정)`
   - 목적: 새로운 기본 모델 설정 기능을 문서에 추가 반영

2. **argument-hint 추가 및 정렬**
   - `model` 서브커맨드 추가 (기본 모델 설정 기능 활성화)
   - 전체 서브커맨드를 알파벳 순서로 정렬
   - 목적: 사용자가 `/` 자동완성을 통해 새 기능 발견 가능하도록 개선

### 동기화 기준

- 커밋 `eaee13c`: feat: lk-setup 기본 모델 설정 기능 추가 — model 서브커맨드로 세션 모델 조회 및 변경
- 이전 버전 `3.5.11`의 미반영 상태를 `3.6.0` 릴리스 사양으로 정렬

## 결과

- 변경된 라인: 2줄 (`description` 1줄, `argument-hint` 1줄)
- 파일 상태: ✓ 수정 완료
- 테스트: 마켓플레이스 재등록 시 `lk-setup` 플러그인 기능 표시 확인 필요
